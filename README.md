# 시니어잡 (SeniorJob) — 시니어 단기 일자리 매칭 플랫폼

> 면접 없이, 당일 입금. 시니어와 고용주를 연결하는 스팟워크 서비스

---

## 빠른 시작

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:5173 열기

---

## 기술 스택

- React 18 + Vite
- CSS Modules (전역 디자인 토큰: src/index.css)
- React Router DOM v6
- lucide-react (아이콘)

---

## 프로젝트 구조

```
src/
├── App.jsx                 # 라우터 (screen 상태로 페이지 전환)
├── index.css               # 전역 CSS 변수 (디자인 토큰)
├── data/
│   └── mockData.js         # 샘플 일자리·지원자 데이터
├── pages/
│   ├── LandingPage.jsx     # 시작 화면 (시니어/고용주 선택)
│   ├── SeniorHome.jsx      # 시니어 홈 (일자리 목록·이력·프로필)
│   ├── EmployerHome.jsx    # 고용주 홈 (공고 등록·지원자 관리)
│   ├── JobDetail.jsx       # 일자리 상세 + 신청
│   └── QRComplete.jsx      # QR 퇴근 + 입금 완료 화면
└── components/
    ├── JobCard.jsx         # 일자리 카드 컴포넌트
    └── BottomNav.jsx       # 하단 네비게이션
```

## 주요 기능

### 시니어 화면
| 화면 | 설명 |
|------|------|
| 랜딩 | 시니어/고용주 역할 선택 |
| 홈 | 카테고리 필터 + 급구/일반 일자리 목록 |
| 상세 | 급여·시간·업무 내용 + 신청 버튼 |
| QR 완료 | 퇴근 QR 스캔 시뮬레이션 + 입금 확인 |
| 이력 | 과거 근무 내역 + 별점 |
| 프로필 | 내 정보·선호 조건·뱃지 |

### 고용주 화면
| 화면 | 설명 |
|------|------|
| 대시보드 | 채용 통계 + 오늘 지원자 목록 |
| 공고 올리기 | 폼 작성 + 급구 옵션 + 시급 자동 계산 |
| 공고 관리 | 등록 공고 현황 + 매칭 상태 |

## 다음 개발 단계 (Claude Code에서 이어서 개발)

### 1단계 — 백엔드 연결
```
- Supabase 또는 Firebase로 실제 DB 연결
- 일자리 CRUD API
- 사용자 인증 (카카오 로그인)
```

### 2단계 — 위치 기반 매칭
```
- 카카오맵 API 연동
- 현재 위치 기반 거리순 정렬
- 지도에서 일자리 탐색
```

### 3단계 — 실시간 알림
```
- 새 공고 푸시 알림
- 매칭 확정 알림
- 입금 완료 알림
```

### 4단계 — 결제 연동
```
- 토스페이먼츠 또는 카카오페이 연동
- QR 기반 자동 정산
- 세금계산서 자동 발행
```

### 5단계 — 통계 대시보드 (대회 출품용)
```
- 통계청 KOSIS API 연동
- 고령화 추이 차트
- 시니어 고용 현황 시각화
- 서비스 필요성 근거 수치
```

## 통계 활용대회 연계 포인트

이 서비스는 다음 통계 데이터를 근거로 설계됐습니다:

- **69.4%** 고령층(55~79세) 장래 근로 희망률 (역대 최고, 2025)
- **73.4세** 평균 희망 근로 연령
- **54.4%** 근로 희망 이유 1위: 생활비 부족
- **86만원** 월평균 연금 (최소 생활비 136만원의 63%)
- **20.3%** 65세 이상 고령인구 비중 (2025, 초고령사회 진입)

출처: 통계청 경제활동인구조사 고령층 부가조사 2025 / 2025 고령자 통계
